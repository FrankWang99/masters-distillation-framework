## 🎯 一句话核心价值

> **不是学大师"做什么"，而是学大师"为什么这样想"。**

大多数人读大师书，只停留在"方法论"层（第一层）。这个框架提供**八层蒸馏法**，帮你从方法论一路挖到人生原点，真正把大师智慧内化成自己的东西。

## 📚 八层蒸馏框架详解

| 层级 | 名称 | 核心问题 | 输出形式 |
|:---|:---|:---|:---|
| **第一层** | 方法论 | 他们做什么、怎么做？ | 文档/工具/操作手册 |
| **第二层** | 底层思维 | 他们为什么这样想？ | 内心判断/直觉 |
| **第三层** | 终极判断标准 | 他们面对一切决策时，只问自己一次的那个问题是什么？ | 简短内心自问 |
| **第四层** | 第一性原理 | 支撑他们思维方式的那个终极假设是什么？ | 底层逻辑框架 |
| **第五层** | 元能力 | 他们有什么样独特的能力？ | 行动清单/技能图谱 |
| **第六层** | 元认知 | 他们如何审视自己的起心动念？ | 内心追问/笔记 |
| **第七层** | 人生使命 | 他们为什么而活？ | 内化驱动力 |
| **第八层** | 人生原点+核心痛苦 | 他们的使命从什么样的关键时刻出发？ | 故事/案例/情感共鸣 |

**推论链条**：
```
方法论 ← 底层思维 ← 终极判断标准 ← 第一性原理 ← 元能力 ← 元认知 ← 人生使命 ← 人生原点+核心痛苦
```

## 🎯 适用人群

| 人群 | 痛点 | 本框架能帮什么 |
|:---|:---|:---|
| **知识博主/内容创作者** | 读完大师书，不知道怎么转化成自己的内容 | 用八层框架拆完，直接出文章/视频脚本 |
| **咨询师/教练** | 学了很多方法论，但不会"内化为自己的东西" | 自我蒸馏后，形成自己的判断标准和话术 |
| **创业者/产品经理** | 想向优秀创业者学习，但只学到表面 | 挖到第一性原理和人生原点，才能真正"学其神" |
| **AI Agent 开发者** | 想让AI具备"大师思维"，不知道怎么建模 | 八层结构可以直接转化为prompt工程或知识库结构 |
| **学术研究者** | 做思想史/人物研究，需要系统性分析框架 | 八层提供标准化的分析维度 |

## 🚀 快速开始

### 1. 安装 SKILL（WorkBuddy 用户）

```bash
# 下载 SKILL 文件
wget https://raw.githubusercontent.com/FrankWang99/masters-distillation-framework/main/SKILL.md

# 在 WorkBuddy 中安装
/install-skill SKILL.md
```

### 2. 非 WorkBuddy 用户

直接参考 `framework/八层框架说明.md`，用这个框架手动蒸馏任何大师。

### 3. 第一次蒸馏

```
请使用 masters-distillation SKILL，蒸馏 [大师姓名]
```

## 📖 使用示例

### 示例1：蒸馏彼得·德鲁克
- **方法论**：目标管理（MBO）+ 五问
- **底层思维**：人不是资源，人是目的
- **终极判断**："这件事，对客户有价值吗？"
- **第一性原理**：企业是社会的器官
- **完整结果**：参见 `examples/德鲁克_八层蒸馏.md`

### 示例2：四位大师对照表
参见 `examples/四位大师_八层蒸馏对照表.md`

### 示例3：自我蒸馏
用同样的八层框架，蒸馏你自己：
- 你的人生原点是什么？（装支架？背债？三重失控？）
- 你的核心痛苦是什么？
- 从原点出发，你的人生使命是什么？

参见 `self-distillation/自我蒸馏指南.md`
