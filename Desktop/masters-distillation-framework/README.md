# 大师思维深度萃取框架 (Masters Distillation Framework)

> 用八层框架蒸馏任何大师、任何思想体系，提炼出可复用的底层思维和操作方法。

[English version below](#english-version)

---

## 🎯 一句话介绍

**不是学大师"做什么"，而是学大师"为什么这样想"。**

大多数人读大师书，只停留在"方法论"层（第一层）。这个框架提供**八层蒸馏法**，帮你从方法论一路挖到人生原点，真正把大师智慧内化成自己的东西。

---

## 📚 八层蒸馏框架

| 层级 | 名称 | 核心问题 |
|:---|:---|:---|
| **第一层** | 方法论 | 他们做什么、怎么做？ |
| **第二层** | 底层思维 | 他们为什么这样想？ |
| **第三层** | 终极判断标准 | 他们面对一切决策时，只问自己一次的那个问题是什么？ |
| **第四层** | 第一性原理 | 支撑他们思维方式的那个终极假设是什么？ |
| **第五层** | 元能力 | 他们有什么样独特的能力？ |
| **第六层** | 元认知 | 他们如何审视自己的起心动念？ |
| **第七层** | 人生使命 | 他们为什么而活？ |
| **第八层** | 人生原点+核心痛苦 | 他们的使命从什么样的关键时刻出发？ |

**推论链条**：
```
方法论 ← 底层思维 ← 终极判断标准 ← 第一性原理 ← 元能力 ← 元认知 ← 人生使命 ← 人生原点+核心痛苦
```

# English Version

# Masters Distillation Framework

> Use the 8-layer framework to distill any master, any ideological system, and extract reusable underlying thinking and operational methods.

---

## 🎯 One-Sentence Introduction

**Don't just learn what masters "do", learn "why they think this way".**

Most people who read masters' books only stay at the "methodology" layer (Layer 1). This framework provides an **8-layer distillation method**, helping you dig from methodology all the way to the life origin, truly internalizing masters' wisdom as your own.

---

## 📚 8-Layer Distillation Framework

| Layer | Name | Core Question |
|:---|:---|:---|
| **Layer 1** | Methodology | What do they do, how do they do it? |
| **Layer 2** | Underlying Thinking | Why do they think this way? |
| **Layer 3** | Ultimate Judgment Standard | What is the one question they ask themselves when facing any decision? |
| **Layer 4** | First Principles | What is the ultimate assumption supporting their thinking? |
| **Layer 5** | Meta-Capability | What unique capabilities do they have? |
| **Layer 6** | Meta-Cognition | How do they examine their own thoughts and intentions? |
| **Layer 7** | Life Mission | Why do they live? |
| **Layer 8** | Life Origin + Core Pain | From what key moment does their mission depart? |

**Inference Chain**:
```
Methodology ← Underlying Thinking ← Ultimate Judgment ← First Principles ← Meta-Capability ← Meta-Cognition ← Life Mission ← Life Origin + Core Pain
```
