# 大师思维深度萃取框架（八层蒸馏法）

**SKILL名称**：masters-distillation  
**版本**：V1.0  
**用途**：为蒸馏任何大师、任何思想体系提供标准化的八层萃取框架

---

## 一、为什么需要八层框架

大多数"学习大师"只停留在第一层——他们说了什么、做了什么。但大师之所以是大师，不是因为他们的方法论有多精妙，而是因为他们看待世界的根本方式与常人不同。

**八层框架的核心逻辑**：从最表面的"怎么做"开始，一层一层剥开，直到挖出那个最底层的"为什么要出发"。

---

## 二、八层框架详解

| 层级 | 名称 | 核心问题 | 蒸馏方法 |
|:---|:---|:---|:---|
| **第一层** | 方法论 | 他们做什么、怎么做？ | 提炼核心框架、工具、流程 |
| **第二层** | 底层思维 | 他们为什么这样想？ | 找出他们看待问题的根本视角 |
| **第三层** | 终极判断标准 | 他们面对一切决策时，只问自己一次的那个问题是什么？ | 凝练为一句可自问的判断句 |
| **第四层** | 第一性原理 | 支撑他们思维方式的那个终极假设是什么？ | 找出他们看待世界的元视角 |
| **第五层** | 元能力 | 他们有什么样独特的能力？ | 识别他们最核心的认知能力 |
| **第六层** | 元认知 | 他们如何审视自己的起心动念？ | 揭示他们"思考自己的思考"的方式 |
| **第七层** | 人生使命 | 他们为什么而活？ | 找到他们穷尽一生要做的事 |
| **第八层** | 人生原点+核心痛苦 | 他们的使命从什么样的关键时刻出发？他们最根本的痛苦是什么？ | 找到那个改变他们一生的转折点 |

---

## 三、八层推论链条

**推论逻辑**：每一层都是下一层的前提，每一层都让你离大师的内核更近一步。

```
方法论 ← 底层思维 ← 终极判断标准 ← 第一性原理 ← 元能力 ← 元认知 ← 人生使命 ← 人生原点+核心痛苦
```

**推论说明**：
1. **第一层→第二层**：方法论背后一定有支撑它的思维方式
2. **第二层→第三层**：底层思维最终会凝练成一个终极判断标准
3. **第三层→第四层**：终极判断标准背后，是对世界的根本假设（第一性原理）
4. **第四层→第五层**：第一性原理会转化为大师身上最突出的认知能力（元能力）
5. **第五层→第六层**：元能力的背后，是大师对自己思维过程的审视方式（元认知）
6. **第六层→第七层**：元认知的反复训练，最终会指向人生使命
7. **第七层→第八层**：人生使命来自某个改变一生的关键时刻（人生原点），而原点处的最根本痛苦，就是一生使命的燃料

## 四、如何使用这个SKILL

### 4.1 蒸馏新的大师

**步骤**：
1. 收集目标大师的核心方法论和著作
2. 按照八层框架逐层推进，填写蒸馏表格
3. 将蒸馏结果填入"八层对照表"
4. 与自己的资产进行融合（参照ANALYSIS-004的融合方式）

**输出**：生成`[大师姓名]_八层蒸馏结果.md`

### 4.2 自我蒸馏

**步骤**：
1. 回溯自己的人生原点（如：装支架？背债？三重失控？）
2. 识别核心痛苦（不想让下一个老板走你走过的弯路？）
3. 从原点出发，明确人生使命
4. 反向推导：元认知 → 元能力 → 第一性原理 → 终极判断标准 → 底层思维 → 方法论

**输出**：生成`自我_八层蒸馏结果.md`

### 4.3 服务嵌入

**步骤**：
1. 将蒸馏出的大师智慧嵌入服务流程
2. 示例（参照SOP-006三位一体服务流程）：
   - 开场用稻盛和夫（动机至善建立信任）
   - 诊断用爱迪思（正常vs异常化解焦虑）
   - 交付用查兰（执行三要素确保落地）
   - 内核用王阳明（致良知校准方向）

**输出**：更新SOP文件，增加大师智慧嵌入话术

## 七、SKILL使用技巧

1. **从外层到内层**：一定要从第一层（方法论）开始，逐层深入，不要跳层
2. **验证每层**：每完成一层，问自己："这一层能解释上一层吗？"
3. **找到痛苦**：第八层（人生原点+核心痛苦）是最关键的，没有痛苦作为燃料的使命，很难持续
4. **立即试用**：蒸馏完立即写出"明天就能用的一句话术"，不要等"以后再用"

## 八、常见问题

**Q1：我可以先蒸馏自己，再用这个框架蒸馏大师吗？**  
A：可以。先自我蒸馏，能帮你建立"识别底层思维"的敏感度，再蒸馏大师会更快更准确。

**Q2：如果某位大师的资料很少，怎么办？**  
A：先完成你能完成的层级，缺失的层级标注为"待补充"。随着你继续研究，逐步完善。

**Q3：这个框架只能用于人物吗？**  
A：不是。你可以蒸馏一个思想体系（如：Buddhism、Stoicism）、一个组织（如：Toyota Production System）、甚至一个时代精神。
